# バイオインフォマティクス・プロトコル集

[English](README.md) | 日本語

> この文書は AI（Claude Opus 4.8）によって `README.md` から翻訳されました。正式な原文は英語版です。誤りや不自然な表現に気づいた場合は、英語版を参照してください。

バイオインフォマティクスでよく行う作業のための、実践的なチュートリアル集です。
これらのチュートリアルは、**プログラミングや計算機科学の正式な前提知識を必要としません**が、
分子生物学にある程度なじみがあることを前提としています。論理、集合論、線形代数、統計といった
数学的なトピックにいくらか親しんでいると役立ちますが、必須ではありません。

各チュートリアルは番号付きのフォルダにまとめられており、順番に進めていくことを想定しています。
教材は Jupyter ノートブックで、概念や関連する演習を説明する `README` が付属しています。

## はじめに

初めての方は、まず [チュートリアル 0](0_Setting_up_your_workstation/README.md) から始めて、
`conda`、Python、Jupyter をセットアップしてください。ワークステーションの準備ができたら、
[チュートリアル 1](1_Basic_python/README.md) に進み、Jupyter Lab でノートブックを開きましょう。
